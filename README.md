# DesafioSantanderRansomwareKeylogger

	Esse desafio tem como intuito mostrar de forma básica como age e funciona um ransomware e um keylogger utilizando simulações controladas com a linguagem de programação python.

Ransomware Simulado: Foram criados arquivos de teste no processo. O ransomware ele sequestra os arquivos pessoais do alvo e encripta, depois o golpista exige um valor para descriptografar os arquivos.
	No teste realizado o script gera uma chave de criptografia e salva depois carrega a chave e a salva, é realizada a criptografia em um único arquivo, encontra arquivos para criptografar. É mandada uma mensagem de resgate ao alvo.

Keylogger Simulado: Foi realizado scripts que capturam tudo que é digitado no teclado e armazenado em um arquivo de modo furtivo para que o alvo não saiba que está sendo submetido a colhimento de dados. O script roda em segundo plano sem que o alvo saiba que está enviando sem o seu consentimento dados digitados, tanto na web como no whatsapp e etc.

Estratégias para evitar os malwares: A proteção é um conjunto de técnicas e hábitos como:


    • Fazer backups.
    • Evitar privilégios de acesso.
    • Utilizar antivírus com IA.
    • Firewall sempre ativo.
    • Atualizar sempre o S.O.
    • Fazer sandboxing.
    • Aprender sobre engenharia social para não cair em golpes e phishings.
      	
Minhas considerações foram que o desafio mostrou de forma prática como agem as pessoas com alguma ferramenta privilegiada e como é simples obter dados para fins maliciosos e criminais.
